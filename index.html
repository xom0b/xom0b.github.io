Ransack

September - December 2016 | University of Utah






Ransack is a 2-4 player couch competitive burglary brawler built in Unity 5

Notable features include a robust platforming system: running, jumping, dashing, wall sliding/jumping, tackling. A throwing system based off a visceral flicking motion with the right analog stick. A unique art style inspired by 1930s cartoons featuring film grain, bloom, and chromatic abberation.

Ransack was built with a team of 2 designers, 1 artist, and 1 programmer (myself). I am responsible for all 3,500+ lines of code on the project excluding a few third party packages (handling Xbox controller input, hue shift shaders, and a rudimentary character controller). A steam released is planned for Summer 2017

PixelYen

April 2016 | University of Utah

PixelYen is a drawing application built in C++ using Qt and Magick++.

PixelYen provides a suite of tools for drawing sprite animations for video games. The user draws a set of frames using the drawing tools and can export them as an animated gif or export them as individual pngs. PixelYen's tools include a pencil/eraser tool, a lighten/burn tool, and a fill tool. PixelYen also features layers for every frame. Users can create, delete and edit layers that act as their own independent canvases. PixelYen also offers an Undo/Redo functionality, allowing users to undo and redo their pencil strokes.


Space Jam

April 2016 | University of Utah

Space Jam is an educational game built in C++ using Qt, SFML, and Box2D.

Aimed at kids in elementary school, Space Jam is a simple physics-based trivia game that quizzes kids on questions about the planets in the solar system. Each planet is a level with a set number of questions. Space Jam has a scoring system that saves progress to a server. Students can log in and start off where they left off and try and improve their previous scores. Teachers can log in to view their student's progress.


AgCubio

December 2015 | University of Utah

Multiplayer game based off the popular browser game Agar.io built using C#, Windows Forms, and MySQL.

AgCubio features custom screen-scrolling, physics, and collisions. Client and server built using windows forms and C#'s asynchronous server sockets. Metadata is saved to a MySQL server and accessible through a web page hosted by the game server.
